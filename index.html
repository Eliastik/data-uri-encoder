Data URI Encoder
Le fichier de style « style.css » est manquant. Il est important pour le fonctionnement de Data URI Encoder. Assurez-vous qu'il soit présent dans le même dossier que le fichier « index.html » et réessayez. Dans le cas échant, re-téléchargez cette application.
Javascript est désactivé ! Ce programme fonctionnant grâce à Javascript, vous ne pourrez pas l'utiliser. Cliquez ici pour savoir comment activer javascript.
Data URI Encoder est partiellement incompatible avec votre navigateur car celui-ci ne supporte pas l'API Canvas. Mettez-le à jour, puis réessayez. Vous pouvez cependant utiliser l'encodeur de documents HTML.

Ce programme vous permet de convertir des images et des textes sous la forme d'une URI encodée en Base64. Cela permet, par exemple, d'intégrer une image directement dans du code HTML sans fichier séparé, ou encore de convertir un document HTML entier en le rendant accessible seulement à l'aide d'une URI. Choississez l'image sur l'ordinateur OU entrez le chemin absolu de l'image. Vous pouvez aussi utiliser une adresse URL vers une image sur un site web, mais il se peut que le site en interdise l'accès aux scripts. Pour convertir un document HTML en entier, entrez son code dans l'onglet « Document HTML ».

Lisez le README.txt pour plus d'informations sur la compatibilité et autres. Programme créé par Eliastik (voir le fichier LICENCE.txt). Découvrez Javascript hqx, un programme créé par Eliastik permettant d'agrandir des images à l'aide du programme HQX (utilisé dans la plupart des émulateurs de console de jeu vidéo) directement sur votre navigateur web préféré.
